# 🌱 AgroWave — Smart Farming Intelligence Platform

**Hackathon Project | Smart Agriculture | India**

Full-stack Next.js 14 platform giving Indian farmers real-time soil intelligence, weather forecasts, and live market prices.

---

## 🚀 Deploy to Vercel (2 minutes)

```bash
# 1. Install dependencies
npm install

# 2. Test locally
npm run dev   →  http://localhost:3000

# 3. Deploy
npm install -g vercel
vercel login
vercel --prod
```

That's it. No API keys needed. No environment variables required.

## 📦 Pages

| Page | Route | Features |
|---|---|---|
| Landing | `/` | Hero, features, crop calendar, testimonials, market preview |
| Dashboard | `/dashboard` | Live farm map, plot cards, charts, weather strip, quick actions |
| Field Monitor | `/field-monitor` | Soil data, NPK editor, moisture slider, add/delete plots |
| Weather | `/weather` | 7-day forecast, hourly chart, farm advisory cards |
| Market | `/market` | Live prices, P&L calculator, mandi rates, price alerts |
| Inventory | `/inventory` | Full CRUD, low-stock alerts, category filters |
| Calendar | `/calendar` | Monthly view, event scheduling, upcoming list |
| About | `/about` | Project details, tech stack, problem/solution mapping |
| Contact | `/contact` | Working form, FAQ accordion |

## 🎨 Design System

- **Colors**: Brand green (#10B981), Sky blue (#3B82F6), Amber (#F59E0B), Rose (#F43F5E)
- **Background**: Clean white + #F0FDF8 canvas — NO dark colors
- **Font**: DM Sans (body) + JetBrains Mono (data)
- **Cards**: White, border-radius 16px, subtle shadow

---

## 🛠 Tech Stack

- **Framework**: Next.js 14 (App Router)
- **State**: Zustand with localStorage persistence
- **Charts**: Recharts
- **Animations**: Framer Motion
- **Styling**: Tailwind CSS
- **Language**: TypeScript
